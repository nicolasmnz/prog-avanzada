C es un lenguaje de programación, creado por Dennis Ritchie en el año 1970. C es conocido por su velocidad y eficiencia en el uso de memoria del computador, por lo que es usado en general para aplicaciones complejas, sistemas operativos, etc.

Esto lo hace un lenguaje muy distinto comparadco con Python (que se vio previamente en Programación). Por eso, esta guia les hara una introducción a los conceptos y el lenguaje de C para poder aplicarlo a lo largo del curso.
